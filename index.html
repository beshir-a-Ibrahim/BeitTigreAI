üìö Overview

The Tigre language, spoken in Eritrea and eastern Sudan, is the closest living relative to Classical Geez ‚Äî a liturgical and historical language of immense cultural and religious significance. While Geez is no longer spoken, Tigre remains vibrant, with growing digital content on platforms like:

  • Wikipedia
  • Mozilla Common Voice (Tigre)
  • Tatoeba (Tigre sentences)
  • Radio, TV, newspapers, and literature

Despite this vitality, Tigre has been underrepresented in Natural Language Processing (NLP). This project aims to close that gap by providing open-access data, models, and tools for researchers, developers, and the Tigre-speaking community.

Additionally, because of its linguistic proximity to Geez, advances in Tigre NLP can serve as a bridge to understanding and potentially revitalizing Geez.

üß© Data

We have compiled a diverse corpus from multiple sources to support robust model training:

Type Description Sources
1. Speech Audio recordings in Tigre Common Voice, field recordings, radio archives
2. Speech/Text Aligned audio-transcript pairs Common Voice, broadcast subtitles
3. Text Monolingual written Tigre Books, news, Wikipedia, social media, religious texts
4. Parallel Data (Translation) Bilingual sentence pairs (e.g., Tigre‚ÄìEnglish, Tigre‚ÄìArabic) Tatoeba, translated documents, crowdsourced efforts

Note: All data is curated, cleaned, and made available in standard formats (e.g., .txt, .csv, .jsonl, .wav).

ü§ñ Trained Models

We leverage state-of-the-art frameworks to develop accessible, high-performance models for Tigre:

Model Description Framework / Base Model
1. Machine Translation Translate between Tigre and other languages Meta's NLLB / Sonar
2. Automatic Speech Recognition (ASR) Speech-to-text for Tigre Meta's Wav2Vec2-BERT
3. Text-to-Speech (TTS) Synthesize natural-sounding Tigre speech rhasspy/piper
4. Large Language Model (LLM) Generative AI for Tigre text Llama 3.2-based, fine-tuned (in progress)

Note: The LLM is currently under development. Early checkpoints will be released soon.

All models are available for download and inference via Hugging Face or direct links.
